# 1. 프로그램 설계서 (5점)

## 1. 프로젝트 주제
교사용 퀴즈 출제 프로그램
- 사용자의 정보를 입력합니다.
-배운 내용을 퀴즈로 복습하며 정답을 입력합니다.
-문제가 끝나면 총점,정답률을 집계하여 격려의 메세지와 함께 보여줍니다.
## 2. 핵심 기능 목록
(구현할 기능들을 순서대로 나열하세요.)
1.  교사가 문제,보기,정답 입력 후 저장
2.  학생으로부터 문제의 답 입력받기
3.  입력된 답과 정답을 비교하기
4.  총점과 정답률, 격려 메세지 출력

## 3. 데이터 구조 설계
(프로그램에 필요한 핵심 변수, 배열을 설계하세요. 필요하면 구조체(struct)를 사용할 수도 있습니다.)

**방법 1: 배열 사용 (기본)**
```c
char question[100][200]; // 문제 문장 저장
int options[100][5][100]; //각 ㅂ문제의 보기 저장 (최대 5개)
int answer[100]; //각 문제의 정답 번호 저장
int user_answer[100] //사용자가 입력한 답 저장
int score; //점수저장
int total_questions; //총 문제 개수
char student_name[50] //학생 정보
```
